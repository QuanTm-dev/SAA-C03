- IAM access keys are long term crendentials - a credential that does not change regularly or rotate.
- IAM users can have a maximum of 2 access keys.
- Access keys can be created, deleted, activated and deactivated.
- Access keys are formed from 2 part:
  - Access key ID (similar to username)
  - Secret access key (similar to password)
- Rotating access keys meaning creating a new access key, updating aapplications to use the new keys, then deleting the old access key.
- Only root user and IAM users can use access keys.
  **Note**: HAVING ACCESS KEYS FOR THE ROOT USER IS NOT RECOMMENDED.
