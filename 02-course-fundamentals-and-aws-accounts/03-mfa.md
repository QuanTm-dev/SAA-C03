# What is MFA?

- MFA stands for Multi-Factor Authentication.
- Factor is something you use to verify your identity.
- There are 4 common factors:
  - Something you know: username, password
  - Something you have: MFA device
  - Something you are: fingerprint
  - Location: which network you are connecting from
    => MFA means using more than one factor to verify your identity.

# Why MFA is important?

- If your username and password are leaker, an attacker can impersonate you and access your AWS resources.

# How AWS MFA works?

- When MFA is activated, a secret key and associated information are generated. These needed to be entered into the MFA application. Typically, this is done by scanning a QR code within the MFA application.
- A MFA application can hold multiple MFA devices. Each MFA device generate a new code for every period of time. Each device is associated with a different account/service.
- The next time you sign in to AWS, you will need to enter both username/password and the code generated by the MFA device.
