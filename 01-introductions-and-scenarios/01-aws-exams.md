# AWS Cert Overview:

- There are 4 main difficulty levels:
  1. Foundational (Practional) => You should skip this cert
  2. Associate => Beginner should start with the certs in this level, especially SA Associate as it is the foundation for every other Associate certs
  3. Professional
  4. Specialty
- There are 2 cert categories:
  - Role based (You should start with these first):
    - SA: SA Associate, SA Professional
    - Operations: SysOps Administrator, Devops Engineer Professional
    - Developer: Developer Associate, Devops Engineer Professional
  - Specialty (These come last): These certs narrow to a aspect, for example security, Networking, etc...

# AWS Exams Overview:

## Associate

- Format: Multiple choices, multiple answers
- Duration: 130 minutes
- Cost: 150$

## Professtional

- Format: Multiple choices, multiple answers
- Duration: 180 minutes
- Cost: 300$

# AWS Cert Order:

## Traditional route

1. SA Associate
2. SysOps Administrator
3. Devops Engineer Professional
4. SA Professtional
5. Devops Engineer
6. Specialty certs: No order required, it depends on the skills you want to learn

## Stream based route (Recommended)

1. SA Associate
2. SA Professtional
3. Other associate and professional certs are optional: Order as above
4. Specialty certs: No order required, it depends on the skills you want to learn
