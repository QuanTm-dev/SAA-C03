# Kubernetes Terms

- Cluster: A set of machines that work together to run containerized applications.
- Control Plane: The brain of the cluster that manages scheduling, scaling, and cluster state.
- Nodes: Physical or virtual machines where Pods run.
- Pod: The smallest deployable unit in Kubernetes, containing one or more containers that share networking and storage. Pods are ephemeral; one container per Pod is most common.
- Service: A stable endpoint that exposes and load-balances traffic to a set of Pods.
- Job: Runs one or more Pods to completion for batch or one-time tasks.
- Ingress: Defines HTTP/HTTPS routing rules to expose Services outside the cluster.
- Ingress Controller: A component that implements Ingress rules and handles external traffic.
- Persistent Volume (PV): Cluster-managed storage that persists beyond the life of a Pod.
- containerd: A container runtime that pulls images and runs containers on a node.
- kubelet: An agent on each node that ensures Pods are running as defined by the Control Plane.
- Kubernetes API (API Server): The central interface used by all cluster components to communicate.

# Control Plane Components

- kube-apiserver: The front-end of the Control Plane that exposes the Kubernetes API.
- etcd: A distributed key-value store that holds the cluster’s state and configuration data.
- kube-scheduler: Assigns Pods to nodes based on resource availability and constraints.
- kube-controller-manager: Runs controllers that continuously reconcile desired and actual cluster state.
  - Node Controller: Monitors node health and reacts when nodes become unavailable.
  - Job Controller: Ensures Job Pods run until completion.
  - Endpoint Controller: Maintains Service endpoints to match the set of ready Pods.
  - Service Account & Token: Provide Pod identities and credentials for authenticating with the Kubernetes API.
- cloud-controller-manager: Manages cloud-specific resources like load balancers and storage.

# Node Networking

- kube-proxy: Manages network rules on nodes to route traffic to Pods.
