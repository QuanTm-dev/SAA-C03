Cloud is a system that meets the following 5 characteristics:

1. On-demand self-service: Users can provision computing resources as needed without human interaction from the service provider (e.g. you don't need to call someone to get a server).
2. Broad network access: Capabilities are available over the network and accessed through standard mechanisms (e.g. you can access cloud services from any device with an internet connection).
3. Resource pooling:
   - There is a sense of location independence, no control or knowledge over the exact location of the provided resources. (e.g. with AWS, you can choose a region, but not the specific data center).
   - Resources are pooled to serve multiple consumer using a multi-tenant model.
4. Rapid elasticity:
   - Capabilities can be elastically provisioned and releaseed to scale radidly outward and inward with demand.
   - To consumer, the capabilities available for provisioning often appear to be unlimited.
5. Measured service: Resource can be monitored, controlled and reported and billed.
