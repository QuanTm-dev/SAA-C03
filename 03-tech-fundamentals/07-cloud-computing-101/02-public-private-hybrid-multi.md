- Public cloud: A cloud environment that is available to the public.
- Private cloud: A cloud environment that is dedicated to a single organization, run on premises.
- Hybrid cloud: A combination of public and private clouds, allowing data and applications to be shared between them.
  **Note:** Hybrid cloud is not a combination of public clouds and on-premises infrastructure.
- Multi-cloud: The use of multiple cloud services from different providers (e.g. AWS + Azure)
  **Note:** Avoid using tools or services that abstract multiple clouds into a single environment. For example, third-party platforms that let you choose which cloud to provision a virtual machine on. These tools add unnecessary complexity and require the same services to be available across all clouds.
