# Layer 3/4/5 Firewalls (Limitations)

- Can only inspect up to their operating layer
- No visibility into HTTP headers, payloads, or application data => Cannot differentiate between legitimate and malicious application-level content

# Layer 7 Firewall (Application Layer)

- Understands all protocols from layer 1-7
- Can inspect and understand application content (HTTP headers, payloads, DNS queries, etc.)
- Identifies normal vs. abnormal requests
- Can read, modify, block, or tag application data
